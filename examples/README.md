# Examples

This directory contains practical LaTeX examples for linguists, researchers, students, and academic publishers.

The goal of these examples is to provide reusable and pedagogical materials illustrating how LaTeX can be used for scientific writing in linguistics and language sciences.

Examples may include:
- minimal documents;
- scientific articles;
- books and monographs;
- Beamer presentations;
- posters;
- glossed examples;
- IPA transcription;
- syntax trees;
- tables;
- bibliographies;
- journal templates;
- multilingual documents.

---

# Why LaTeX in linguistics?

LaTeX is widely used in linguistics and academic publishing because it provides:
- high typographic quality;
- stable handling of complex documents;
- support for Unicode and IPA;
- automated bibliography management;
- advanced support for glosses and syntax trees;
- editorial consistency across publications.

Many major publishers and journals in linguistics either:
- compose their publications in LaTeX;
- provide official LaTeX templates;
- or accept LaTeX submissions.

# Suggested workflow

Most examples in this repository are designed for:
- XeLaTeX;
- LuaLaTeX;
- Unicode;
- OpenType fonts.

Recommended packages include:
- `fontspec`
- `linguex`
- `tikz-qtree`
- `biblatex`
- `polyglossia`

# Repository philosophy

This repository promotes:
- structured scientific writing;
- editorial automation;
- reproducible academic workflows;
- accessible typographic resources for linguistics.

The examples are intended both:
- as learning materials;
- and as reusable foundations for real academic projects.
